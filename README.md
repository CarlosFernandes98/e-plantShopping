# e-plantShopping

**Descrição do Projeto**  
e-plantShopping é uma aplicação web de comércio eletrônico focada em plantas de interior. Permite aos usuários navegar por diferentes categorias de plantas, adicionar produtos ao carrinho, ajustar quantidades, remover itens e visualizar o total do carrinho. A interface é intuitiva e responsiva, ideal para amantes de plantas que desejam comprar online.

---

## Funcionalidades

- Página inicial com mensagem de boas-vindas e botão "Get Started".  
- Página de listagem de produtos (`ProductList.jsx`) com:
  - Pelo menos 6 plantas por categoria.
  - Pelo menos 3 categorias diferentes.
  - Imagem, nome e preço de cada planta.
  - Botão "Add to Cart" para adicionar produtos ao carrinho e atualizar o contador de itens.  
- Carrinho de compras (`CartItem.jsx`) com:
  - Total do carrinho atualizado dinamicamente.
  - Total por produto baseado na quantidade.
  - Botões para aumentar/diminuir quantidade.
  - Botão para remover itens do carrinho.
  - Botão de Checkout (com alerta “Coming Soon”).
  - Botão "Continue Shopping" para voltar à lista de produtos.
- Navbar fixa em todas as páginas com links para Home, Plants e Cart.
